Common HTML Tags

These are the most commonly used HTML tags and a short description of each.

Block Elemments

Block elements take up their own line and take up the full width of the page.

body

Defines the document's body.

head

Defines info about the document.

html

Defines the HTML of the document.

div

Defines a division or a section in an HTML document.

section

Defines sections in a document, such as headers, footers, etc.

header

Element represents a container for introductory content or a set of navigational links.

main

Tag specifies the main content of a document.

article

Tag specifies independent, self-contained content.

nav

Defines a set of navigation links.

footer

Defines a footer for a document or section. Examples copyright info, sitemap, contact info, back to top link, etc.

p

Defines a paragraph.

aside

Defines some content aside from the content it is place in.

li

Defines a list item.

Inline Elements

Inline elements only take up as much space on a line as they need to. They do not start new lines.

a

This tag defines a hyperlink.

strong

This element makes text bold.

em

This tag is a phrase tag. It renders as emphasized text.

img

This tag embeds an image on a page. You must use a src attribute to pick the image.

span

This tag is used to group inline-elements in a document.
